# Обучение студентов основам Python

### Первое задание: текстовый квест

Сюжет: человек приходит в казино, 
1. На входе его спрашивают, хочет ли он войти, ответы да-нет.
2. Спрашивают, во что хочет играть: рулетка (1), покер (2), однорукий бандит (3)

Рулетка: просим загадать число от 0 до 32, выкидываем случайное. Если угадал, то отлично.
